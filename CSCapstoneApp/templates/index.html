About The Project


This project has been designed for students, professors, and engineers to create and collaborate on real-world projects. Engineers are able to create a project that they want to have finished by a certain date and students can request to volunteer for participating on the project. Not only does this allow the engineer finish the project sooner, but it also gives his/her company a look at possible future employees. The student is also greatly benefited because s/he is able to see what it is like to work on a real-world group project that is often not able to be taught in a classroom setting. Teachers are able to view the projects and who has been working on the projects. This allows them to either assign students to these already created projects or create another version of the project that is more suitable for a classroom setting.

Upon registration, you can select whether you are a student, professor or engineer. Each user type has different rights to of access and interaction with the website. For example, when signed up as an engineer, you are able to create projects, whereas students and teachers can only view and participate on projects given the engineers permission. Anyone is allowed to view the various groups under the 'Search' tab and easy communication between users can be done through the 'Comments' page. Feel free to meander about the site and checkout the available projects.
